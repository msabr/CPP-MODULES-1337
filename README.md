## Build and run

Build and run each exercise from its own folder:

1. Go to a specific exercise directory (example: `CPP Module 03/ex01`)
2. Run `make`
3. Run the produced executable
4. Use `make clean` / `make fclean` when needed

## Notes

- Target standard across modules is generally **C++98**.
- Typical flags used in the modules are: `-Wall -Wextra -Werror`.
